# lignin-dataset


## Monomer / Monolignol
- collection of the main components of lignin structure
  - lignin_sequences.smiles : SMILES file
  - minimized_wrapped.pdb : minimized structure 
  - lignin.pdb : original structure generated from SMILES 

## Dimer 
- collection of the physically acceptable lignin dimers 
  - lignin_sequences.smiles : SMILES file
  - minimized_wrapped.pdb : minimized structure 
  - lignin.pdb : original structure generated from SMILES
